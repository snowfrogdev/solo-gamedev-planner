# Law of Common Region

Elements tend to be perceived into groups if they are sharing an area with a clearly defined boundary.

## Description

The Law of Common Region is a Gestalt principle of visual perception that describes how placing elements within a shared visual boundary (such as a box, background color, or card) causes them to be perceived as a related group. This visual containment is a powerful grouping mechanism that can override other Gestalt principles like proximity or similarity — elements inside a shared boundary are perceived as related even if they differ in appearance or are spaced apart. It's one of the most practical tools in a designer's toolkit for creating clear visual hierarchy and communicating element relationships.

## Key Takeaways

- Common region creates clear structure and helps users quickly understand relationships between elements and sections.
- Adding a border around elements is an effective way to establish common region.
- Defining a distinct background behind elements also creates common region.
- Common region can override other grouping principles like proximity or similarity.

## Origins

The grouping principles (Gestalt laws) originated from Gestalt psychology, which proposed that humans naturally perceive objects as organized patterns and objects — a principle known as Prägnanz. Gestalt psychologists theorized that the mind has an inherent tendency to perceive patterns based on specific organizational rules. The Law of Common Region was formally identified by Stephen Palmer in 1992 as an extension of the original Gestalt principles established by Max Wertheimer and others in the early 20th century. The five foundational categories of grouping principles are: Proximity, Similarity, Continuity, Closure, and Connectedness.

## Practical Implications for UI Design

- **Card-based layouts**: Cards are the most common implementation of common region in modern UI design. Each card contains related information (image, title, description, action) within a defined boundary.
- **Form section grouping**: Group related form fields within visual containers to communicate that they belong together (e.g., "Shipping Address" fields in one container, "Payment Information" in another).
- **Navigation and toolbars**: Contain navigation items or toolbar actions within a shared region to signal they serve a common purpose.
- **Dashboard widgets**: Use bordered or backgrounded containers to separate distinct data displays and help users understand the information architecture.
- **Use restraint**: Before adding borders and colored boxes, ask whether whitespace alone is sufficient. Excessive visual containers create clutter and can paradoxically confuse users rather than clarify.
- **Test assumptions**: Validate during usability testing whether visible boundaries are actually necessary or whether simpler proximity-based grouping achieves the same communication goals.
- **Avoid false stopping points**: Heavy visual containers can create the impression that content has ended, discouraging users from scrolling further.

## Further Reading Insights

**Nielsen Norman Group (Aurora Harley)** — Items within a boundary are perceived as a group and assumed to share common characteristics or functionality. Practical applications include the Windows Print dialog (grouping printer selection, page range, and copy count), tabbed interfaces, and accordion panels. Key caution: boundaries can override other grouping principles, so use them strategically. Always test whether whitespace alone would suffice before adding visual containers.

**Smashing Magazine (Steven Bradley)** — The Law of Common Region is one of the most powerful Gestalt principles for web design because it provides explicit visual grouping that requires no interpretation. Designers should implement it through card-based layouts, modal dialogs, and grouped form sections to establish visual hierarchy and organization. The principle works alongside other Gestalt laws — combining common region with proximity and similarity creates the clearest possible visual communication.

**Scholarpedia** — Gestalt principles describe the brain's tendency to organize visual information into coherent groups. Common region works because the brain interprets shared boundaries as evidence of shared function — a heuristic that evolved to help process complex visual environments efficiently.
